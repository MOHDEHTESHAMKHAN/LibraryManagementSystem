MEK Library Dashboard Project Description The MEK Library Dashboard is a web-based application designed to manage and organize a library's book collection efficiently. This project provides a user-friendly interface for librarians and administrators to add, update, delete, and view books in the library. The dashboard includes features such as data tables for displaying book information, alerts for notifications, and a responsive design for accessibility on various devices.

Features Book Management: Add, update, delete, and view books. Responsive Design: Accessible on desktops, tablets, and mobile devices. DataTables Integration: Enhanced table functionalities like sorting, searching, and pagination. Alerts and Notifications: Display important messages and alerts to users. User Authentication: Secure login and access control for administrators.

Technologies Used Frontend: HTML, CSS (Bootstrap), JavaScript (jQuery, DataTables) Backend: PHP Database: MySQL
